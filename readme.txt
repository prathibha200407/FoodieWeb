??? Food Munch – Food Ordering Web App
-------------------------------------

Food Munch is a simple front-end food ordering website built using HTML, CSS, JavaScript, and Bootstrap.
It includes a working cart system, order summary, location detection, UPI modal, and order history stored in localStorage.


?? Features
-----------

ADD TO CART
- Add items to cart
- Cart saved in localStorage
- “Go to Cart” button opens Order Summary
- Toast notification when item is added

ORDER SUMMARY / CHECKOUT
- View cart items
- Increase / decrease quantity
- Remove items
- Auto subtotal and total calculation
- Delivery or Dine-In selection
- Table number input for dine-in
- Name, phone, address for delivery

LOCATION DETECTION
- Detects your location using browser GPS
- Reverse-geocodes using OpenStreetMap
- Auto-fills delivery address

PAYMENT
- Cash on Delivery
- UPI option with copy-to-clipboard support

ORDER HISTORY
- Saves past orders in localStorage
- Displays them in Your Orders section

EXTRA UI FEATURES
- Story / gallery upload
- Chef quote slider
- Mood filter buttons
- Smooth Bootstrap UI

Then open:
